Quattro Rioni — Cultural Association of Cilavegna

The project was initiated to create a new web presence for the cultural association “Quattro Rioni” in Cilavegna. Founded in 2024, the association did not yet have a website to promote its activities in the town.

This project was important for me because, as a local Councillor in Cilavegna, it is my priority to support and promote associations that aim to develop the local community.

Project Goals

  • Build a clean and responsive mini-site to present the association.
  • Provide an event calendar, gallery, and membership form.
  • Use tools covered in class: HTML, CSS, Bootstrap 5, JavaScript.

Website Structure

Homepage: contains the title, hero image, presentation, and a call to action to immediately join the association; in the center: about us, with a summary of the activity of the association, and a calendar made by Google Calendar to show upcoming events; at the bottom: a footer with e-mail and phone number, links to social pages, and a map to easily find Cilavegna and the association’s address.

Gallery contains images of past events (Bootstrap carousel).
Subscribe: a form to collect name and email to become a member of “Quattro Rioni”.

All pages are connected with a menu for easy navigation.

Folder Organization

Technology I Used

  • HTML for the structure of the web pages.
  • CSS to customize the pages and apply styling.
  • Bootstrap 5 for flexboxes, image carousel, icons and subscription form.
  • JavaScript for interactivity at the end of the subscription process, displaying a “Thank you for subscribing to our community!” confirmation message.

Long-Term Plans

  1. Implement a fully functional mailing list.
    Connect the subscription form to a real mailing-list service, such as Mailchimp, to automate sign-ups and email campaigns.

  2. Add an online donation page.
    Create a donations page with online payment options (PayPal, Stripe) to support community events and charitable initiatives.

  3. A more professional event management system.

  4. Add a “Blog” or “News” section.
    Provide updates, event summaries, and cultural content for the community.

Things I Learned

  • How to organize a website clearly (homepage, sections, footer)
  • How to use Bootstrap components and customize them with CSS
  • How to make the site responsive for mobile and desktop (even if this was one of the hardest challenges)
  • How to use a simple JavaScript for interactivity
